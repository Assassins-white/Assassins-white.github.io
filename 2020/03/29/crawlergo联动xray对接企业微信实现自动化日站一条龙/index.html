crawlergo联动xray对接企业微信实现自动化日站一条龙

Posted on 2020-03-29

所谓的丧，不过是闲到发慌

介绍

crawlergo

crawlergo是360天相的动态爬虫模块

目前支持以下特性：

  • 原生浏览器环境，协程池调度任务
  • 表单智能填充、自动化提交
  • 完整DOM事件收集，自动化触发
  • 智能URL去重，去掉大部分的重复请求
  • 全面分析收集，包括javascript文件内容、页面注释、robots.txt文件和常见路径Fuzz
  • 支持Host绑定，自动添加Referer

xray

长亭洞鉴安全评估系统

主要特性有：检测速度快、支持范围广、代码质量高、高级可定制、安全无危胁

下载

crawlergo_x_XRAY：https://github.com/timwhitez/crawlergo_x_XRAY

crawlergo：https://github.com/0Kee-Team/crawlergo/releases

xray：https://github.com/chaitin/xray/releases

使用

把下载好的 crawlergo.exe 放到 crawlergo_x_XRAY 的目录下，把 crawlergo 文件夹删除，否则后面运行可能会出现权限不足等问题

启动xray被动扫描 ./xray.exe webscan --listen 127.0.0.1:7777 --html-output test.html

脚本默认配置为 127.0.0.1:7777 如果修改xray监听端口就要在 crawlergo_x_XRAY 的launcher.py 文件里面同步修改 proxies 设置

launcher.py 的cmd变量中的 crawlergo 是爬虫配置,特别注意浏览器路径,如果电脑性能不好的话,可以把 -t 线程数调低一点，默认是 20

把目标填进 crawlergo_x_XRAY 的 target.txt 文件里面，一行一个，前面需要加上协议

Xray开启被动扫描，使用 python3 运行 launcher.py

生成的 sub_domains.txt 为爬虫爬到的子域名, crawl_result.txt为爬虫爬到的url

这样我们就实现了crawlergo跟xray的联动了，我们再对接企业微信机器人，实现出洞后自动提醒

对接企业微信机器人

使用企业微信，创建群聊，添加机器人

复制机器人的 webhook 地址

使用官方提供给我们的代码，稍微更改一下

然后把脚本放到服务器上面去

使用 nohup python xray.py & 命令

让他在后台持续运行

有什么模块没安装的，pip install xxx 装上就完事

然后 xray 使用 Webhook 输出

xray.exe webscan --listen 127.0.0.1:7777 --webhook-output http://你服务器IP:5000/webhook
